# dotfiles

Some of my personal (config-) files.

## Installation using GNU Stow

```shell
zypper in stow
git clone git@github.com:nodeg/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow $APPLICATION (e.g. `stow zsh`)
```
